Speedman.comのREADME
筆者:三上航世
5人1組のチームで作ったゲーム(筆者はメインプログラマ担当)

操作方法
キーボード(ゲームパッドが繋がっている場合、キーボードは強制終了のESC以外反応しない)
移動					:W,A,S,D
ジャンプ				:SPACE
ジップライン掴み、離し	:SPACE
ジップライン中の移動	:W,S(Wが前、Sが後ろ)
カメラ移動				:マウス移動
ポーズ					:ESC
ポーズ中の選択			:方向キーかW,S
決定					:エンターキー
強制終了				:ESC(タイトル、リザルト、ゲームオーバー画面のみ)

ゲームパッド(Direct Inputのみ。また、ゲームの途中でつなげた場合、1回exeを落とさないとゲームパッドは反応しない)
移動					:左スティック
ジャンプ				:A
ジップライン掴み、離し	:A
ジップライン中の移動	:左スティック(前に倒すと前に、後ろに倒すと後ろに進む)
カメラ移動				:右スティック
ポーズ					:START
ポーズ中の選択			:方向キー
決定					:B

ゲームの仕様
3Dプラットフォームゲーム
1人用
ステージの奥にあるゴールを目指す
ゴールした時のクリアタイムと残機によってスコアが変わる
初期残機は10
下に落ちると残機が1減る
残機がなくなるとゲームオーバー
コースの途中にはチェックポイントがあり、そこを通った後に落ちるとそこからやり直しになる

特殊オブジェクト
ジャンプパッド	:乗ると大ジャンプをする
ジップライン	:縄を掴んだ移動をする

コツ
一部ショートカットが可能なので、それを覚える
ジップラインも焦らずに進んで落ちないようにする

坂に関して
ステージ後半にある坂ですが、元々は実装できなさそうなら実装しなくていいと言われていたものでした。
ですが、基本的な当たり判定としてまず、XZの2つの軸を見て、真上から見たエリア内に入っているかを処理し、
入っていてかつY軸を見るときに、posと角度とXZ軸方向にどれだけ進んでいるかを見たものを掛け合わせて、坂として処理しました
