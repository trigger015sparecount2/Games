AllTargetsBreakのREADME
筆者:三上航世
1人で作ったゲーム。第2作目

操作説明
キーボード
移動			:A,D
手裏剣投げ		:I
手裏剣投げの方向:W,A,S,D
ジャンプ		:J
ポーズ			:P
ポーズ中の選択	:W,S
決定			:J
強制終了		:ESC

ゲームパッド(Direct Inputのみ。また、ゲームの途中でつなげた場合、1回exeを落とさないとゲームパッドは反応しない)
移動			:左スティック
手裏剣投げ		:RB
手裏剣投げの方向:右スティック
ジャンプ		:A
ポーズ			:START
ポーズ中の選択	:方向キー
決定			:B

ゲームの仕様
2Dアクションゲーム
1人用
ステージにある的を手裏剣で全て壊すとクリア
クリアするまでの時間を競う
特殊ギミックとして、左右に押すブロック、乗ると動く足場、氷の足場、ワープホールがある
一部はギミックを上手く使ったりしないと壊せない

コツ
とにかく何度もやって覚える
一部ショートカットが出来るので、それを覚える
