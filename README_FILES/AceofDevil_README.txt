ポーズ			:エンターキー

ゲームの仕様
2Dシューティングゲーム
1人用
敵を倒すか弾を打ち消すとスコアが上がる
ステージ最後にいるボスを倒すとゲームクリア
クリア時のスコアを競う
自分が出来る攻撃の種類は全5種類
敵機は雑魚が4種、ボスが1体
攻撃は押しっぱなしで可能
攻撃するときにそれぞれのエネルギーが消費される
敵の弾丸は全て打ち消し可能
打ち消すするには反対の属性の弾、あるいは風属性のバリアをぶつける
炎(赤)と氷(水色)、雷(黄色)と水(青)が反対の属性になってる
打ち消すと相手の弾を消しながらエネルギーを吸収できる

攻撃の種類
炎:正面にいる敵を追尾する弾を1発撃つ。連射力が高い
氷:正面3方向に貫通する弾を1発ずつ計3発撃つ。攻撃力は少し低い
雷:正面に弾を1発撃つ。弾が敵に当たると、周りにいる敵や水属性の弾に放電する。攻撃力は高いが、連射力が低い
水:入力した方向に水のレーザーを放つ。扱いにくい代わりに攻撃力は高め。
風:自身に風のバリアを纏う。攻撃力はかなり低い代わりに全ての属性の弾を相殺できる

コツ
敵の戦闘機の色と撃ってくる弾の属性の色は一致しているので、敵の色を見て属性を素早く変えて撃つ
ラスボスのみ様々な属性を撃ってくるが、基本的に炎を使うと素早く倒せる
弾を打ち消すとエネルギーと共にスコアが貰えるため、弾は極力打ち消す
打ち消し方によって貰えるスコアが変わる。風だとあまり貰えない
攻撃と属性変化を交互に素早くやると、雷でもかなりの連射力で撃てる。その代わり、エネルギーもかなりの速度で減る
