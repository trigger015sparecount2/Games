特殊兵装		:I

ポーズ中の選択	:W,S
決定			:J

ゲームの仕様
2Dシューティング
1人用
敵を倒すとスコアが貰える
全部で3ステージあり、3ステージ目のボスを倒すとゲームクリア
ステージが切り替わるごとに自機のダメージが回復し、特殊兵装の弾数が1以下だと2まで回復する
自機は2種類、特殊兵装はそれぞれ3種類、計6種類
敵機は4種、ボスは3種類
攻撃は1回ボタンを押すごとに1発弾が出る
特殊兵装は1回使うごとに一定時間のクールダウンが発生するとともに1発使う

自機と特殊兵装
F-15SM
頑丈な代わりに少し遅い
特殊兵装も守り寄り
特殊兵装
UAV					:自機の左右に小型無人戦闘機を配置する。自機がショットを撃つとUAVも撃つ
マルチロックミサイル:最大6体の敵をロックオン、6発ミサイルを放つ。5体以下しかいない場合、同じ敵に2発以上放たれる
バリア				:自機を守るバリアを張る。バリアは3回ダメージを受けると消える。時間では消えない
					 また、バリアの展開中にバリアを張ることは出来ない

SU-27E
装甲が薄い代わりに速い
特殊兵装は攻撃寄り
特殊兵装
レールガン			:自機正面に小型の素早い貫通弾を1発放つ
フレアミサイル		:1体の敵をロックオン、当たると大爆発を起こすミサイルを1発放つ
レーザー			:自機正面にレーザーを放つ

コツ
普通の攻撃の弾数は無限なのでとにかく連射しつつ、偏差を意識する
特殊兵装は意外と敵が落とすので躊躇しないで使う
