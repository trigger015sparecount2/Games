ゲームの仕様
3DTPS
1人用
敵を倒すとスコアが貰える
次々倒していくとチェインが繋がり、さらにスコアが貰える
ボスを倒すとクリア
クリア時のスコアを競う
攻撃は射撃のみ
撃つときには、操作キャラが向いている方向にいる敵に向かって自動で照準を合わせてくれる
撃つたびに弾数を消費し、無くなるとリロードするまで撃てなくなる
敵はボス1体、雑魚3体
アイテムはステージ上に落ちており、拾うと使える
アイテムは、使うと持っている数が1つ減る代わりにHPが全回復する

コツ
敵を倒すとチェインが繋がるので、チェインを切らさないようにしてボスのHPも削りつつ雑魚を倒していく
