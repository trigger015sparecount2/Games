概要:1人で作った3Dアクションゲーム。第3作目

操作方法
キーボード
移動			:W,A,S,D
攻撃			:J
ジャンプ		:I
ロックオン		:E
カメラ移動		:マウスを左クリックしながら移動(ドラッグ)
カメラのリセット:マウスホイール押し込み
強制終了		:ESCキー

ゲームパッド(Direct Inputのみ。また、ゲームの途中でつなげた場合、1回exeを落とさないとゲームパッドは反応しない)
移動			:左スティック
攻撃			:B
ジャンプ		:A
ロックオン		:RB
カメラ移動		:右スティック
カメラのリセット:右スティック押し込み

ゲームの仕様
1人用
敵は一度に50体まで出る
1体倒すごとに追加される
敵が合計100体出てくるので全て倒す
100体倒すまでにやられるとゲームオーバー
主人公の攻撃は敵との位置関係によって種類が変わるので追尾力が高め
連続で攻撃するとコンボが繋がる
敵の種類は近接攻撃役と遠距離攻撃役の2種類
敵は一度に3体まで攻撃してくる
ロックオンはカメラの向きのよって変わる

コツ
敵に攻撃を当てたら即逃げるというヒット&アウェイを心がけるとあまりダメージをくらわないで倒せる
